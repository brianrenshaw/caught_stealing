To connect your Yahoo Fantasy league, follow these steps:

  1. Register at developer.yahoo.com/apps/create/
  2. Set app type to Installed Application
  3. Check Fantasy Sports under API Permissions
  4. Copy Client ID and Client Secret to your .env file
  5. Add your YAHOO_LEAGUE_ID (from your league URL)
  6. Restart the server
